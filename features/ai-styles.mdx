## Premium vs Free Styles

<Info>
  **Premium Styles** are marked with a lock icon and require unlocking style packs. Free styles are available to all users immediately.
</Info>

Premium style categories include:
- All Formula 1 styles
- Complete John Wick collection (8 styles)
- Full Superhéroes pack (7 styles)
- Peaky Blinders series (4 styles)

## Usage Statistics

Each style tracks usage count to show popularity. The most-used styles appear in the "Top Estilos" section, helping users discover trending transformations.

## Dynamic Style Management

Styles can be dynamically managed through the admin dashboard:
- Toggle individual style visibility
- Mark styles as premium/free
- Upload custom style images
- Track real-time usage analytics

<Note>
  Style availability updates in real-time across all user sessions using Supabase's real-time subscriptions.
</Note>
